# `( )`

- _singular_: parenthesis
- _plural_: parentheses
- `(`: _opening_ parenthesis
- `)`: _closing_ parenthesis

```js
'use strict';

/* --- control flow checks --- */

if (false) {
} else if (false) {
}

while (false) {}

for (let nothing of '') {
}
```

```js
'use strict';

/* --- function calls --- */

alert('an alert');
prompt('a prompt');
confirm('a confirm');

console.log('a console log');

let name = 'Timlin';
let bigName = name.toUpperCase();
console.log(bigName);
```
