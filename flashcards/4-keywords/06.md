This **keyword** is used to _end_ a loop immediately, whether or not the
condition is truthy or falsy.

# `break`

```js
'use strict';

let userInput = null;

while (true) {
  userInput = prompt('do not cancel');

  // execute the body until the user clicks "ok"
  if (userInput !== null) {
    break;
  }
}

alert(userInput);
```
